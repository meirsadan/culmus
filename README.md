# Culmus.js

Access the live website at https://culmus.xyz/

This instructional website demonstrates how to write Hebrew letters using a broad nib pen, in a casual formal style.

It was coded by Meir Sadan, with typeface designed by Daniel Grumer. Created as an educational aid for the Department of Visual Communication, Bezalel.
